# Project Modules

Each visible project card is loaded from its own folder:

- `sol/project.js`
- `aurora/project.js`
- `aether/project.js`
- `fake-news/project.js`

Edit the text, tags, metrics, architecture notes, demo ideas, and media placeholders inside that project's `project.js`.

Put videos, screenshots, and architecture diagrams inside that project's `assets/` folder. Then update the matching `mediaSlots` entry in `project.js` with the right label or description.
